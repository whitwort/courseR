Let's plot a normal distribution in blue

x <- seq(-4, 4, length = 200)
y <- dnorm(x, mean = 0, sd = 1)
plot(x, y, col = "blue")

Two plots on one graph

x <- seq(-4, 4, length = 200)
y1 <- dnorm(x, mean = 0, sd = 1)
y2 <- dnorm(x, mean = 1, sd = 1)
plot(x, y1, type = "l", col = "blue")
lines(x, y2, col = "red")
