Bio 285: Introduction to Systems Biology

Welcome to the course website for Bio 285: Introduction to systems biology! Here you'll find links to all of the key resources for this class. The course syllabus, assignments and links to licensed primary research articles can be found on our sakai.wlu.edu site.

Software

The tiny bit of code used to build this site goes a long way because:

  • courseR is the backend R script that builds the html pages from markdown source (see /source). It uses knitr (GPL) to handle literate programming, highlight (GPL2) to handle html class annotation on R code fragments, and markdown (GPL3) to compile the rest of the document.
  • reveal.js is used as the html slide show engine (MIT)
  • HTML5BoilerPlate is used as the starting point for our html and css (Unlicense).
  • jQuery is used extensively for DOM hacking. (MIT/GPL).
